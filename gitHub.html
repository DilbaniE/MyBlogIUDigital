Cómo utilizar Git - Tutorial Detallado

Introducción

Git es un sistema de control de versiones distribuido que permite a los desarrolladores rastrear cambios en su código y colaborar con otros. En este tutorial, aprenderemos cómo inicializar un repositorio local, crear ramas y conectar un repositorio local con GitHub.

1. Instalación y Configuración de Git

1.1 Descarga e Instalación

Para Windows, descarga Git desde el sitio oficial:

Descargar Git para Windows

1.2 Configuración Inicial

Después de la instalación, configura tu nombre de usuario y correo electrónico:

git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"

2. Inicialización del Repositorio Localmente

Para comenzar a usar Git en tu proyecto:

  1. Abre la terminal en la carpeta del proyecto.
  2. Ejecuta el comando: git init
  3. Git creará un nuevo repositorio en esa carpeta.

2.1 Agregando Archivos al Staging Area

Para preparar los cambios para el commit:

git add .

2.2 Realizando el Primer Commit

Para guardar los cambios en el repositorio:

git commit -m "Mensaje descriptivo del commit"

3. Creación y Manejo de Branches

Las ramas en Git permiten trabajar en nuevas funcionalidades de forma aislada.

3.1 Creando una Nueva Rama

git branch nombre-de-la-rama

3.2 Listando las Ramas

git branch

3.3 Cambiando entre Ramas

git checkout nombre-de-la-rama

3.4 Fusionando Ramas (Merge)

Para incorporar los cambios de una rama a otra:

git checkout rama-destino
git merge rama-origen

4.1 Creando un Nuevo Repositorio en GitHub

Conclusión

Este tutorial te ha guiado a través de los pasos básicos para usar Git, desde la inicialización de un repositorio local hasta la conexión con GitHub. Practicar estos comandos te ayudará a familiarizarte con el flujo de trabajo de Git y mejorar tu gestión de proyectos de software.
